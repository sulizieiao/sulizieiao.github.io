## markdown 基本语法

```reStructuredText
代码: ```python （代码块），...python （行内代码）
公式：$行内公式$, $$行间公式$$
列表：+，-，1.
换行：1. 行尾加入两个空格  2.写成代码块形式  3.空一行  4. 插入`<br>`
```
